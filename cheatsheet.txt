Inicializar un repositorio
git init

Mostrar el estado de todo el repositorio
git status

Prepara un archivo para un comit
git add

Guardar un archivo en el repositorio
git commit

Mostrar todos los commit que se hicieron
git log

Traer los cambios hechos en el repositorio
git pull

mandar los cambios hechos hacia el repositorio
git push
